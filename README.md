## How to restore and push

1. Install Git on your local machine.
2. Open the repository folder:
   - `C:\Users\Admin\Documents\tech---premi\tech---premi-main`
3. Initialize or reinitialize Git if needed:
   - `git init`
4. Add remote if missing:
   - `git remote add origin https://github.com/swastikthegenius-cloud/tech---premi.git`
5. Stage and commit:
   - `git add .`
   - `git commit -m "Reconstruct project from live deployment"`
6. Push to GitHub:
   - `git branch -M main`
   - `git push -u origin main`

If your repository already has a remote branch, replace `main` with the correct branch name.
