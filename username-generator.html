How it works

Choose a style, choose a format, then set how long you want the usernames to be. Use one word for tighter names, tag style for endings like xd12 or -=123, and mixed for a blend.

The AI button tries to create more creative usernames. If AI is unavailable, the page falls back to a quick formatter so the tool still works.
